# WiFi Analysis Dashboard

WiFi 데이터 분석 웹 애플리케이션

## 빠른 시작

### Vercel로 배포 (가장 쉬움!)

1. GitHub에 이 레포지토리 업로드
2. https://vercel.com 접속
3. "New Project" 클릭
4. GitHub 레포지토리 선택
5. Deploy 클릭
6. 완료! 🎉

URL이 자동 생성됩니다: `https://your-project.vercel.app`

### 로컬 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev

# 브라우저에서 http://localhost:5173 열기
```

## 기능

- CSV 파일 업로드
- WiFi 데이터 통계 분석
- 실시간 시각화 (6가지 차트)
- 반응형 디자인

## 기술 스택

- React + Vite
- Recharts (차트)
- Tailwind CSS (스타일)
